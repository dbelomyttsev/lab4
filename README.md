# Лабораторная работа №4

#### Описание
Это приложение Flutter извлекает и отображает список криптовалют. Данные извлекаются из удаленного API с помощью пакета Dio.

#### Используемые технологии
* Dart
* Flutter Framework
* Flutter SDK
* Dart Packages
* Dio
